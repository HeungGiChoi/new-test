## 파이썬 교육
 - 결과값이 있는 함수를 호출할땐, 값을 변수에 넣어서 호출하면 된다.
 - 함수에서 내부에 지역변수가 있으면 지역변수를 사용한다.
 그러나, 내부에 지역변수가 없고 전역변수만 있으면 전역변수를 사용
한다.
 - 전역변수를 활용하여 함수 내부에서 연산을 반복하면, 전역변수 내부
수가 변한다.
 - 슬라이싱으로 거꾸로 시작. 맨끝에서 맨 처음까지 출력하려면,
[ : : -1] 이다.
원래는 맨 처음부터 맨 마지막까지인데, 스텝 위치가 -1 이므로
맨 처음 시작 위치는 맨 끝이 되고, 맨 마지막 위치가 맨 처음으로
바뀐다.
 - 맨 처음부터 맨 끝까지 출력은 굳이 [ : : 1] 할 필요 없이, 
  [ : : ]만 하면 된다.
 - 리스트 함수
  * insert(요소, 값) 함수를 이용해서 리스트의 원하는 인덱스 위치에
원하는 값을 추가할 수 있다.
  * pop(인덱스)는 리스트의 특정 인덱스에 있는 값을 삭제
  인덱스를 생략할 경우, 리스트의 맨 마지막 요소를 삭제
 - for문은 내가 지정한 횟수만큼 반복, while문은 특정 조건이 만족하는
 동안 반복
 - for문의 range()에서 값이 하나만 들어가면, 0부터 값-1까지 숫자를 
 만듦
 - range()에 10, 0, -1을 차례로 넣으면, 10부터 1까지 역순으로 출력
  * 1, 10이면 1부터 9이듯, -1이므로 10부터 0이면 10부터 0까지가 
   아니라 10부터 1까지임
